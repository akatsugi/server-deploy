# Server Deploy IDEA Plugin

This is a Gradle-based IntelliJ IDEA plugin project.

## Toolchain

- JDK: `D:\Environment\Java`
- Gradle: `D:\Environment\gradle-8.14.3`

## Scripts

- `build-plugin.bat`: package plugin zip
- `test-plugin.bat`: run Gradle tests
- `run-ide.bat`: start sandbox IDEA with the plugin loaded

## Output

The packaged plugin zip is generated under `build/distributions`.
